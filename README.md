# Fee Creds

Live Demo: 

https://fee-creds.onrender.com

## Overview

Fee Creds is a web application developed by Fredi, designed to offer a user-friendly platform for managing and exploring credit-based services.

## Features

- Clean and responsive UI
- Fast performance and smooth navigation
- Hosted and always online via Render

## Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: No backend used
- Hosting: Render.com

## Getting Started

To run the project locally:

1. Clone the repository:
   ```
   git clone https://github.com/FezChat/fee-cred.git

3. Navigate to the project directory:
   ```
   cd fee-cred

5. Install dependencies:
   ```
   npm install

7. Start the server:
   ```
   npm start
